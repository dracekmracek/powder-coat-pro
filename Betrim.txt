# BETRIM - DOKUMENTACE PROJEKTU

## ZÁKLADNÍ INFORMACE
- **Název projektu:** Betrim - Práškové lakování
- **Popis:** Webová prezentace pro firmu zabývající se práškovým lakováním kovů
- **Jazyk webu:** Čeština
- **Cílová skupina:** Průmysloví i soukromí zákazníci hledající služby v oblasti práškového lakování

## TECHNOLOGIE

### Frontend
- **Framework:** React 18.3.1 s TypeScript 5.5.3
- **Build systém:** Vite 5.4.14
- **Routing:** React Router DOM 6.26.2
- **UI komponenty:** 
  - Vlastní komponenty založené na Radix UI (kompletní sada komponent)
  - Stylování pomocí Tailwind CSS 3.4.11
  - Animace a efekty pomocí Framer Motion 12.5.0
- **Formuláře:** Formsubmit.co
- **State management:** React Query (TanStack Query) 5.56.2
- **Vizualizace dat:** Recharts 2.12.7

### Deployment a hostování
- **Hosting:** Miramo spol. s.r.o.
- **Deployment:** Automatizovaný pomocí vlastního skriptu (build-deploy.sh)
- **Verzování:** Git

## STRUKTURA PROJEKTU
```
src/
├── components/      # Opakovaně použitelné UI komponenty
│   ├── sections/    # Komponenty reprezentující jednotlivé sekce stránky
│   └── ui/          # Základní UI prvky (Shadcn UI + vlastní)
├── hooks/           # Vlastní React hooks
├── lib/             # Utility funkce, konfigurace (např. utils.ts)
├── pages/           # Komponenty reprezentující jednotlivé stránky/route
│   └── legal/       # Právní stránky (Cookies, Privacy, Terms)
├── App.tsx          # Hlavní App komponenta (routing, providery)
├── main.tsx         # Vstupní bod aplikace, renderování Root
└── index.css        # Globální CSS styly (primárně Tailwind base, components, utilities)
```

## JAK PROJEKT SPUSTIT A UPRAVOVAT

### Lokální vývoj
1. Naklonování repozitáře
2. Instalace závislostí: `npm install`
3. Spuštění vývojového serveru: `npm run dev`
4. Server běží na http://localhost:8080

### Sestavení a nasazení
1. Sestavení produkční verze: `npm run build`
2. Nasazení na FTP: `npm run deploy` (používá vlastní skript build-deploy.sh)
